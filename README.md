## 📦 Installation

### Quick Install

1. **Install BetterDiscord** — If you haven't already, download and install BetterDiscord from [betterdiscord.app](https://betterdiscord.app)

2. **Clone or Download** — Download this repository or clone it:
   ```bash
   git clone https://github.com/NovaCreations/BetterDiscord.git
   ```

3. **Place Addons**
   - **Plugins**: Copy `.plugin.js` files to `%APPDATA%\BetterDiscord\plugins\`
   - **Themes**: Copy `.theme.css` files to `%APPDATA%\BetterDiscord\themes\`

4. **Enable** — Enable addons in BetterDiscord settings

> 💡 Check the [guides](./guides/installation/) folder for detailed installation instructions for each addon.
